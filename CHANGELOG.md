All notable changes to this marketplace will be documented here.

## [1.0.0] - 2026-02-17

### Added
- Initial marketplace setup with proper structure
- Complete plugin template demonstrating all capabilities
  - Skills, agents, hooks examples
  - MCP and LSP server configurations
  - Hook scripts (pre-submit, post-submit)
- Comprehensive documentation
  - Creating plugins guide
  - Using plugins guide
  - Plugin template documentation
- Repository structure for multiple plugins
- GitHub Actions workflow for validation
- Marketplace manifest at `.claude-plugin/marketplace.json`
